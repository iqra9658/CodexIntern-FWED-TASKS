# React Router Demo

This is a minimal Vite + React project demonstrating client-side routing using `react-router-dom` (v6).

Quick start (PowerShell):

```powershell
cd "C:\Users\Moin Bhai\react-router-demo"
npm install
npm run dev
```

Open the URL shown by Vite (typically `http://localhost:5173`) and use the navigation links.
